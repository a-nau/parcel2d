Compontents

Our work has 4 code components, described below.


Image Scraping  

First you need to gather potential relevant images. If your object of interest appears frequently in image search results, feel free to test our image scraping tool. It supports
  • Google Images
  • Bing Images
  • Baidu Images
  • Yahoo Images

Image Selection  

This repository helps you to
  • automatically remove the background of images
  • pre-select images (based on general criteria)
  • crop images
  • generate split files for training
