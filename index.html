Compontents

Our work has 4 code components, described below.


Image Scraping  

First you need to gather potential relevant images. If your object of interest appears frequently in image search results, feel free to test our image scraping tool. It supports
  • Google Images
  • Bing Images
  • Baidu Images
  • Yahoo Images

Image Selection  

This repository helps you to
  • automatically remove the background of images
  • pre-select images (based on general criteria)
  • crop images
  • generate split files for training

Dataset Generation  

Once you have an image pool consisting of objects of interest, distractor objects and background images, you can generate your own dataset! This repository is a modified (and extended) version of debidatta/syndata-generation and supports
  • easy configuration of image composition (number of objects, degree of occlusion)
  • augementation (rotation and scaling)
  • different blending methods

CNN Training  

Once you created your dataset, you probably want to test it as well. This repo contains
  • Notebooks for dataset visualization
  • (Tested) code for training with Detectron2
  • Evaluation routines
  • Notebook for qualitative evaluation
